QuoteGenerator
========

"QuoteGenerator" is a WordPress plugin to generate custom quotes.
In this case, it creates custom quotes around products pack of our web agency.


**REQUIREMENTS**

- PHP 7.1
- WORDPRESS 4.9.8 or higher
